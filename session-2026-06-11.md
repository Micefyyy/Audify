# Session — June 11, 2026

## Summary
Full UI overhaul of the Audify music streaming app — library cards, queue system, smart shuffle, inline lyrics, NowPlaying redesign, and Home page content.

## Changes Made

### UI Cleanup (Library)
- **`src/pages/Library.tsx`** — Smaller `PlaylistCard` (4:3 aspect, `text-xs`, `p-2`, `rounded-xl`), 4-column grid (`grid-cols-4 gap-1.5`), compact header/tabs, smaller floating "+" button, tighter `TrackRow` layout
- **`src/styles/globals.css`** — Added `.scrollbar-none` utility class

### Queue System
- **`src/store/playerStore.ts`** — Added `addToQueue`, `removeFromQueue`, `clearQueue`, `reorderQueue` actions; `recentlyPlayed[]` auto-tracking (last 20 plays)
- **`src/pages/Queue.tsx`** — New page at `/queue` showing current track + upcoming queue with play/remove controls
- **`src/App.tsx`** — Added `/queue` route
- **`src/pages/NowPlaying.tsx`** — Top-right button navigates to Queue
- **`src/pages/Search.tsx`**, **`src/pages/PlaylistDetail.tsx`**, **`src/pages/Library.tsx`** — Added "Add to queue" (`ListPlus`) button on track rows

### Home Page Redesign
- **`src/pages/Home.tsx`** — Time-based greeting, Now Playing banner with animated bars, Recently Played horizontal scroll, Browse by Genre grid, "Start Exploring" CTA for new users
- **`src/pages/Search.tsx`** — Reads/writes `?q=` URL param for genre browsing from Home

### NowPlaying — Smaller Artwork + Inline Lyrics
- **`src/pages/NowPlaying.tsx`** — Artwork capped at 32vh, added scrollable synced lyrics area filling remaining space (auto-scroll to current line, tap to seek), removed separate Lyrics page link; added `flex-shrink-0` to all fixed sections
- **`src/services/lyricsService.ts`** — Existing (unchanged, fetches from lrclib.net)

### Smart Shuffle + Playlist Shuffle
- **`src/store/playerStore.ts`** — Added `smartShuffle` state, `toggleSmartShuffle()`, `fetchSmartRecommendations()` helper; auto-fetches similar tracks from Piped API when queue runs low
- **`src/services/audioService.ts`** — Added `getRecommendations(seed, excludeIds)` searches by artist/title
- **`src/pages/PlaylistDetail.tsx`** — Added "Shuffle" button (randomizes order before play)
- **`src/pages/NowPlaying.tsx`** — Smart shuffle toggle (Sparkles icon, grouped with regular shuffle)

### Playback Speed & Spam Protection
- **`src/store/playerStore.ts`** — `html5: true` for instant streaming, `isLoading: false` set before `play()`, added `if (isLoading) return;` guard in `play()` to prevent double-play from spam clicking
- **`src/components/Player/MiniPlayer.tsx`** — Removed "Buffering…" text, removed `disabled={isLoading}` on play button
- **`src/pages/NowPlaying.tsx`** — Removed "Loading…" text

### Layout Fix (MiniPlayer Persistence)
- **`src/components/Layout.tsx`** — Removed `AnimatePresence` around MiniPlayer; wrapper stays in DOM permanently, animates via framer-motion `animate` prop instead of mount/unmount

### Artist Profiles
- **`src/services/audioService.ts`** — Added `getArtistTracks(artistName)` searches Piped and returns tracks filtered by artist name
- **`src/pages/Artist.tsx`** — New page at `/artist/:name` showing circular artist image (from top track artwork), artist name, "Top tracks" list with play buttons, skeleton loading state
- **`src/App.tsx`** — Added `/artist/:name` route
- **`src/pages/Search.tsx`**, **`src/pages/Library.tsx`**, **`src/pages/PlaylistDetail.tsx`** — Artist names in all `TrackRow` components are now clickable buttons that navigate to the artist page (`stopPropagation` to avoid triggering track play)
- **`src/pages/Home.tsx`** — Artist names in Recently Played cards are also clickable links to artist page
